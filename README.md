## 👋 Обо мне

Меня зовут **Velvet** — русскоязычный разработчик на **Python**.

- 🔧 Люблю работать с **API-системами** — проектировать, интегрировать, разбираться, как они устроены изнутри
- ⚙️ Увлекаюсь **backend-разработкой** — то, что происходит «под капотом», интересует меня больше всего
- 🐳 Собираю и деплою проекты в **Docker**, настраиваю CI/CD, чтобы код доезжал до продакшена сам
- 🤖 Пишу **Discord-ботов** и интегрирую их с бэкендом — от команд до общей базы данных с сайтом
- 🌱 Потихоньку изучаю **React**, во многом опираясь на помощь ИИ в процессе обучения
- 💬 Открыт к общению и совместной работе над интересными задачами

<br>

## 🚧 Сейчас работаю над

Бэкендом и Discord-ботами для **Guild World** — галереи Discord-серверов с тегами кланов: FastAPI-сайт + мультисерверный бот с системой тикетов, автодеплоем на VDS через GitHub Actions и общей SQLite-базой между сайтом и ботами.
